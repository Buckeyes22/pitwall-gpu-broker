# Support

Need help? Here's where to get it:

- **Bug reports & feature requests** — open an issue using the [issue templates](https://github.com/buckeyes22/pitwall-gpu-broker/issues/new/choose)
- **Questions & general discussion** — start a thread on [GitHub Discussions](https://github.com/buckeyes22/pitwall-gpu-broker/discussions)
- **Project docs** — see the [SDLC documentation](https://github.com/buckeyes22/pitwall-gpu-broker/tree/main/docs/sdlc) and the [README quickstart](https://github.com/buckeyes22/pitwall-gpu-broker/blob/main/README.md)

Support is best-effort and no response-time SLA is offered. Never post API keys,
database URLs, webhook secrets, customer inputs, or other sensitive logs. Use
the private process in [SECURITY.md](SECURITY.md) for vulnerabilities.
